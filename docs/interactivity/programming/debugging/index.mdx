# Debugging

## Logging
Based on what you wish to log, you can determine the importance which influences the visual response.

## Code
```cs
Log.Info( "This is an Info" );
Log.Warning( "This is a Warning" );
Log.Error( "This is an Error" );
```

## Overview
To gain a basic overview, how much Infomational, Warnings or Errors have been throw.
